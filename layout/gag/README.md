# Gagauz language (Gagauz dili) — `gag`

Keyboard layout for the **Gagauz language** (ISO 639-3: `gag`), Turkic language family.

**Language codes**:

* **ISO 639-3**: `gag`
* **ISO 15924**: `Latn`

```yaml
displayNames:
  en: Gagauz
  ru: Гагаузский
  gag: Gagauz dili
```

## Gagauz iOS

* gag-3-rows.yaml (Mobile-optimized)

If needed, the second row can be centralized if the dot symbol is removed from it.

## Gagauz macOS

* gag-macos.yaml (Standard Latin-based layout)

## Details

### Алфавит / Alphabet

31 letters (Latin script):

a, ä, b, c, ç, d, e, ê, f, g, h, ı, i, j, k, l, m, n, o, ö, p, r, s, ş, t, ţ, u, ü, v, y, z.

### Названия клавиш / Key names

```yaml
keyNames:
  space: Boşluk
  return: Giriş
  return-alts:
    search: Ara
    go: Geç
    send: Yolla
    join: Kat.
    route: Marşrut
    next: İleri
    continue: Devam et
    done: Bitti
  emergency: SOS
  cancel: İptal
  undo: Kullanma
  redo: Tekrarla
```
