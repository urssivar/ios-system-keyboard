# Кайтагская раскладка (iOS/macOS)

**Язык**: Кайтагский (_Хайдаҡӏла_)

**ISO 639-3**: `xdq`

**Письменность**: Кириллица (31 символ, 42 буквы)

**Основа**: Русская раскладка ЙЦУКЕН. Минимальные изменения для сохранения мышечной памяти.

## Алфавит

**Буквы**: `а б в г ғ д е ж з и й к кк кӏ ҡ ҡҡ ҡӏ л м н о п пп пӏ р с т тт тӏ у х ҳ ц цц цӏ ч чч чӏ ш ъ ь я`

**Символы**: `а б в г ғ д е ж з и й к ҡ л м н о п р с т у х ҳ ц ч ш ъ ь я ӏ`

**Неиспользуемые русские буквы**: `щ ф ы ю э ё`

**Репозиторий со спецификацией**: [Urssivar/Script](https://github.com/urssivar/script)

## iOS

**Переключатель языков**: `ҡғҳ` (от _ҡҡуғадеҳ_ "счастье")

**Долгое нажатие**:

- `у` → `ю` `у́`
- `е` → `э` `е́` `ё`
- `ш` → `щ`
- `п` → `ф`
- `а` → `а́`
- `о` → `о́`
- `я` → `я́`
- `и` → `ы` `и́`

## macOS

**Alt-слой** (заменённые русские клавиши):

- `⌥` `ҡ` = `щ`
- `⌥` `ҳ` = `ф`
- `⌥` `ғ` = `ы`
- `⌥` `ӏ` = `ю`

**Dead key**: `´` для ударных гласных. Например: `⌥` `´` + `а` = `а́`.
