# Нанайский язык (На̄ни хэсэни) — `gld`

Раскладка клавиатуры для **нанайского языка** (ISO 639-3: `gld`), тунгусо-маньчжурская семья.

Автор раскладки: **Василий Харитонов**

## Описание

Нанайский язык использует кириллический алфавит на основе русского с добавлением буквы **Ӈ ӈ**. В учебной и научной литературе для обозначения долгих гласных применяется **макрон** (◌̄).

Раскладка включает все буквы нанайского алфавита, а также специфические символы, используемые в лингвистических и учебных текстах: ӡ, ᵸ, ʼ (модификатор), і, ө, њ и другие.

## 4-рядная раскладка

```
̄   І  Ө  Ы  Ь  Ш  Ф  ᵸ  З  ʼ     ← Head (доп. ряд)
Ё  Е  У  Э  Й  В  Б  Д  Ӡ  Г
-  И  О  А  Р  Л  М  Н  Њ  Ӈ
,  .  Я  Ю  Х  С  П  Т  Ч  К
```

Head-ряд содержит буквы, которые в 3-рядной раскладке доступны только через лонгпресс: макрон (◌̄), і, ө, ы, ь, ш, ф, ᵸ, з, ʼ. Благодаря этому в 4-рядной раскладке лонгпрессы перераспределены на другие буквы (г→ҕ, к→ӄ, ӡ→ǯ, х→ӽ, ӈ→ң и др.).

## Особенности раскладки

- **Компактная кириллическая раскладка** — не на основе стандартной русской ЙЦУКЕН, а оптимизирована для нанайского языка
- **Макрон** (◌̄) — ключевой диакритик нанайского, обозначает долготу гласных (а̄, о̄, э̄, ӣ, ӯ, ё̄, е̄, я̄, ю̄)
- **Ӡ ӡ** (U+04E0/U+04E1) — специфическая нанайская буква
- **ᵸ** (U+1D78) — модификаторная буква
- **ʼ** (U+02BC) — модификатор (апостроф), используется в дʼ, нʼ
- **Ԩ ԩ** (U+0528/U+0529) — дополнительная буква кириллицы

## Системные кнопки (keyNames)

Переводы системных кнопок используют русские варианты, что соответствует сложившейся практике двуязычия среди носителей нанайского языка.
