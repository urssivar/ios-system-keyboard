# Nanai Keyboard Layout (gld) / Нанайская раскладка

This directory contains the keyboard layouts for the Nanai language (`gld`).
В этой папке содержатся раскладки для нанайского языка (`gld`).

## 3-row layout (iPhone / iPad) / 3-рядная раскладка
The 3-row layout uses a **9-9-7** grid optimized for modern Nanai orthography.
3-рядная раскладка использует сетку **9-9-7**, оптимизированную для современной нанайской орфографии.

### Layout / Сетка
```
х  р  й  к  п  т  ч  э  у
с  л  в  г  б  д  и  а  о
   ,  .  ӈ  м  н  ’  ̄
```

## 4-row layout (iPhone) / 4-рядная раскладка
Uses a **9-9-9-7** grid (Head-Top-Middle-Bottom). Vowel macrons are moved to longpresses of the head/vowel keys.
Использует сетку **9-9-9-7**. Макроны гласных перенесены в лонгпрессы соответствующих букв.

### Layout / Сетка
```
ш  ь  ж  з  ы  я  ё  е  ю  ← Head
х  р  й  к  п  т  ч  э  у  ← Top
с  л  в  г  б  д  и  а  о  ← Middle
   ,  .  ӈ  м  н  ’  ̄     ← Bottom
```

## macOS Layout / Раскладка macOS
A phonetic-based layout with a powerful **Alt (Option)** layer for specialized characters and vowel macrons.
Фонетическая раскладка с функциональным слоем **Alt (Option)** для спецсимволов и долгот.

- **Alt + Vowel** = Vowel with macron (а̄, о̄, э̄...).
- **Alt + Consonant** = Specialized letter (ҕ, ӄ, ӈ, ӽ, ӡ, ў, і).
- **Alt + ш** = Deadkey for macron (◌̄).

## Key Names / Названия клавиш
Localised Nanai terminology is used for system keys:
Используется нанайская терминология для системных клавиш:
- Space: **Алдаᵸ**
- Return: **Нирухэᵸ**
- Search: **Баогу**
- Done: **Тахамби**
