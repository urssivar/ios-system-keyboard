# **Bashkir (bak) native iOS/macOS keyboards.**

## Bashkir iOS
There are 3 variants: bak-4-rows (primary), bak-3-rows, and bak-3-rows-rus.

3 rows layout hides less frequent letters and makes them accessible via longpress.

4 rows layout is the established Bashkir keyboard. It has been used on Windows since the 2000s and on Intel-based Macs, so most users are already used to it. This is the primary layout for iOS.

The Russian-based version keeps the standard Russian layout and places Bashkir-specific letters on longpress.

Versions sorting for iPhone:
* bak-4-rows.yaml
* bak-3-rows.yaml
* bak-3-rows-rus.yaml

For iPad keyboard versions there is only 1 version, because there is enough space to put Bashkir letters.

## Bashkir macOS
There are three macOS variants:
* bak-macos.yaml (Apple-style)
* bak-macos-pc.yaml (PC-style)
* bak-macos-alt.yaml (Alt/Option)

`caps` and `caps+shift` switch to the Russian layout.

Third one is based on the standard Russian layout. Bashkir-specific characters are accessed using the `Alt/Option` key.
