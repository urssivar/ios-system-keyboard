# Chuvash (chv) iOS/macOS keyboards

**Language**: `Чӑваш чӗлхи (Чӑвашла)`

**Language codes**:

* **ISO 639-1**: `cv`

* **ISO 639-3**: `chv`

## Alphabet

`а ӑ б в г д е ё ӗ ж з и й к л м н о п р с ҫ т у ӳ ф х ц ч ш щ ъ ы ь э ю я`

**Only in loanwords**: `б г д ё ж з о ф ц щ ъ`
