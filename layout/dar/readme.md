# Даргинская раскладка (Дарган мез)

Даргинская раскладка клавиатуры для iOS.

## Особенности
- Основана на стандартной кириллической раскладке.
- Включает символ Палочка (`Ӏ`) для специфических даргинских звуков.
- Широкий набор диакритик и альтернативных символов через долгое нажатие.

## Раскладка (3 ряда)
```
й ц у к е н г ш щ з х
ф ы в а п р о л д ж э
я ч с м и т ь б ю ъ Ӏ
```

## Долгие нажатия
- Ц -> ЦӀ
- К -> КӀ, Кь, Къ
- Е -> Ё
- Г -> ГӀ, Гь, Гъ
- Х -> ХӀ, Хъ, Хь
- П -> ПӀ
- Ч -> ЧӀ
- Т -> ТӀ
- Ь -> Ъ
