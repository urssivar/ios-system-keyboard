# Dargin Keyboard Layouts / Дарган мез (iOS/macOS)

**Author**: `Murtazali Rabadangadzhiev`

**Language**: `Dargwa (Дарган мез)`

**ISO 639-3**: `dar`

## Alphabet

`а б в г д е ё ж з и й к л м н о п р с т у ф х ц ч ш щ ъ ы ь э ю я Ӏ`

**Specific Sounds (Digraphs and Palochka)**:
`гъ гь гӏ къ кь кӏ пӏ тӏ хъ хь хӏ цӏ чӏ`

## iOS

#### Dargin (Standard) / Дарган (стандартлӑ)

`dar-3-rows.yaml`

A compact 3-row layout (**11-11-9**). Optimized for fast access to the Palochka (`Ӏ`) and Hard Sign (`ъ`).

```
й ц у к е н г ш ъ з х
ф Ӏ в а п р о л д ж э
  я ч с м и т ь б ю
```

**Key Features**:
- **Ӏ (Palochka)** is on the main layer instead of `ы`.
- **ъ (Hard Sign)** is on the main layer instead of `щ`.
- **ы** is available via long-press of **Ӏ**.
- **щ** is available via long-press of **ш**.

## macOS

#### Dargin (Standard) / Дарган (стандартлӑ)

`dar-macos.yaml`

```
] 1 2 3 4 5 6 7 8 9 0 - =
  й ц у к е н г ш ъ з х ъ ё
  ф Ӏ в а п р о л д ж э
  я ч с м и т ь б ю /
```

**Key Features**:
- Consistent with the mobile layout for a seamless experience.
- **ы** is available via **Option (Alt) + Ӏ**.
- **щ** is available via **Option (Alt) + ш**.
