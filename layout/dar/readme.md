# Даргинские раскладки / Дарган мез (iOS/macOS)

**Автор**: `Murtazali Rabadangadzhiev`

**Язык**: `Дарган мез (Даргинский)`

**ISO 639-3**: `dar`

## Алфавит

`а б в г д е ё ж з и й к л м н о п р с т у ф х ц ч ш щ ъ ы ь э ю я Ӏ`

**Специфические звуки (диграфы с Палочкой и другие)**:
`гъ гь гӏ къ кь кӏ пӏ тӏ хъ хь хӏ цӏ чӏ`

## iOS

#### Даргинский (стандартный) / Дарган (стандартлӑ)

`dar-3-rows.yaml`

Компактная 3-рядная раскладка (11-11-9). Оптимизирована для быстрого доступа к Палочке (`Ӏ`) и Твердому знаку (`ъ`).

```
й ц у к е н г ш ъ з х
ф Ӏ в а п р о л д ж э
  я ч с м и т ь б ю
```

**Особенности**:
- **Ӏ (Палочка)** вынесена на основной слой вместо `ы`.
- **ъ (Твердый знак)** вынесен на основной слой вместо `щ`.
- **ы** доступна через долгое нажатие **Ӏ**.
- **щ** доступна через долгое нажатие **ш**.

## macOS

#### Даргинский (стандартный) / Дарган (стандартлӑ)

`dar-macos.yaml`

```
] 1 2 3 4 5 6 7 8 9 0 - =
  й ц у к е н г ш ъ з х ъ ё
  ф Ӏ в а п р о л д ж э
  я ч с м и т ь б ю /
```

**Особенности**:
- Повторяет логику мобильной раскладки для единообразия.
- **ы** доступна через **Option (Alt) + Ӏ**.
- **щ** доступна через **Option (Alt) + ш**.
