# Komi (kom) native iOS/macOS keyboards.

## Komi iOS

There are 2 layers: kom-3-rows and kom-4-rows. 

3 rows layout is default one. The middle and bottom rows look similar to the Russian ones. The top row contains keys for the additional Komi symbols "ӧ" and "і". To maintain the 3-row structure, less frequent letters (Ц, Х, Щ, Ё) are accessible via long-press on their phonetic or visual counterparts (Ч, К, Ш, Е). Historical Komi letters are also available through the long-press menu.

4 rows keyboard is designed for high-speed typing without long-press gestures. This version adds a fourth row at the top for less frequent characters, reducing the number of keys per row for better precision and comfort.

### Versions sorting for iPhone:

    * kom-3-rows.yaml
    * kom-4-rows.yaml

iPad Version: Only one layout is provided for iPad. Due to the larger screen dimensions, all Komi and Russian letters fit onto the main layers without the need for character replacement or hidden menus.

## Komi macOS

The middle and bottom rows mirror the standard Russian keyboard. The top row contains keys for additional Komi symbols "ӧ" and "і" replacing less frequent letters (Щ, Х) respectively, these letters are accessible via the Option (Alt). The Option (Alt) is also used to type characters included in the multilingual Komi layout for Windows.

## Komi keyNames

Key names are translated into Komi using natural interface phrasing.
