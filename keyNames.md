# Локализация системных кнопок клавиатуры iOS (Русский язык)

Этот документ фиксирует официальные переводы системных кнопок клавиатуры (`UIReturnKeyType` / `SubmitLabel`) для iOS.

Переводы выверены по системным приложениям (UIKit/SwiftUI) и подтверждены скриншотами с реальных устройств (iOS 17+).

## 1. Итоговая конфигурация (YAML)

Используйте этот список для маппинга клавиш. Он обеспечивает 100% нативное поведение интерфейса.

```yaml
keyNames:
  # --- Основные кнопки действия (Action Keys) ---
  space: Пробел            # С большой буквы (стандарт iOS 15+)
  return: Ввод             # Стандартный Enter
  
  return-alts:
    search: Найти          # Глагол (Action). Не путать с "Поиск" (Placeholder).
    go: Перейти            # Глагол. Используется в браузерах.
    send: Отправить        # Глагол. Используется в мессенджерах (TikTok).
    join: Подкл.           # ВАЖНО: Системное сокращение. Полное слово не влезает.
    route: Маршрут         # Существительное. Контекст: "Построить маршрут".
    next: Далее            # Навигация по полям формы.
    continue: Продолжить   # Переход к следующему шагу между несколькими полями (например в анкетах).
    done: Готово           # Завершение ввода, скрытие клавиатуры.

  # emergency: Экстренный вызов. Это ошибка. Полный текст "Экстренный вызов" используется только в заголовках.
  emergency: SOS         # На экране блокировки на кнопке используется короткое "SOS".

  # --- Зона конфликтов (Внимание!) ---
  cancel: Отменить         # Глагол. Выход из режима (Search/Nav Bar/SOS). Чтобы перестать вводить текст – клавиатура скроется.
  undo: Отменить           # Глагол. Отмена действия (Ctrl+Z). На Русском языке совпадает с Cancel! На вашем языке может быть совершенно другое слово!
  redo: Повторить          # Глагол. Возврат действия (Ctrl+Shift+Z), возврат отмененного действия. На вашем языке может быть совершенно другое слово!
```

## 2. Доказательства и логика (Reference)

Ниже приведены обоснования переводов, основанные на гайдлайнах Apple и скриншотах интерфейса (iOS 17+).

### ✅ Space (Пробел)
* **Перевод:** `Пробел`
* **Источник:** Подтверждено на системной клавиатуре.
* **Нюанс:** Начиная с iOS 15 пишется с **Большой буквы**.

### ✅ Search (Найти)
* **Перевод:** `Найти`
* **Источник:** Подтверждено синей кнопкой в Spotlight и Safari.
* **Логика:** Используется глагол ("Что сделать?").

### ✅ Emergency (Экстренный вызов)
* **Перевод:** `Экстренный вызов`
* **Источник:** Заголовок системного экрана SOS.
* **Нюанс:** Строка очень длинная. Используется преимущественно на экране блокировки.

### ✅ Join (Подкл.)
* **Перевод:** `Подкл.`
* **Источник:** Меню ввода пароля Wi-Fi в Настройках.
* **Логика:** Полное слово "Присоединиться" (14 символов) шире, чем экранная кнопка iPhone в портретной ориентации. Apple использует официальное системное сокращение.

### ✅ Send (Отправить)
* **Перевод:** `Отправить`
* **Источник:** Приложение TikTok (сообщение), старые версии iMessage.
* **Логика:** Глагол, обозначающий действие отправки данных.

### ✅ Route (Маршрут)
* **Перевод:** `Маршрут`
* **Источник:** В старых Apple Maps или в других приложениях с поездками или поиском путей. Предлагается, когда выбраны точки поездки.
* **Логика:** Используется существительное. Подразумевается контекст: "Построить [Маршрут]". Глаголы "Проложить" или "Маршрутизировать" слишком длинные и сложные.

### ✅ Go (Перейти)
* **Перевод:** `Перейти`
* **Источник:** Адресная строка Safari.
* **Логика:** Стандартный глагол навигации.

### ✅ Done (Готово)
* **Перевод:** `Готово`
* **Источник:** Кнопка скрытия клавиатуры (Toolbar) или сохранения изменений. Странно, но я нашел в приложении Uber при вводе конечного адреса.
* **Логика:** Наречие, обозначающее завершение процесса/статус.

### ✅ Emergency (SOS)
* **Перевод:** `SOS`
* **Источник:** Экран блокировки (кнопка внизу слева).
* **Логика:** Английская кнопка "Emergency" переведена как "SOS". Полная фраза "Экстренный вызов" используется только как заголовок экрана, так как она слишком длинная для кнопки.

### ✅ Cancel (Отменить)
* **Перевод:** `Отменить`
* **Источник:**
    1. Экран "Изменение остановки" в Apple Maps (кнопка слева сверху).
    2. Экран "Экстренный вызов" (SOS).
* **Логика:** В современных версиях iOS Apple перешла на использование глагола "Отменить" для кнопок действия, уходя от существительного "Отмена".

### ✅ Undo / Redo (Отменить / Повторить)
* **Перевод:** `Отменить` / `Повторить`
* **Источник:** Меню редактирования текста (Callout Bar) и панель шорткатов iPad.
* **Логика:** Глаголы действия.
* **⚠️ Важно:**
    * **Undo** ("Отменить") совпадает по переводу с **Cancel**.
    * **Redo** ("Повторить") может быть ложно истолковано пользователем как "Сделать заново" (Repeat), а не "Вернуть отмененное".
    * На Русском языке своя особенность! На вашем языке могут быть совершенно другие слова!
